## Product: Game that is meant to help with Ap Chem studying

## Timeline

Midwinter break use to learn tools more, mostly stuff we will use in the game/API

Ideally by March 8 up to the 15th have the game part done as well as the API
* AIM FOR MULTIPLE LEVELS IF POSSIBLE (3 or 4+ levels, ideal: 9 for each unit of AP Chem)
* AIM FOR MULTIPLE UNITS BEING ADDED TO THE API

Buffer days from April 1st to 13th for MVP

After MVP complete as much as the beyond MVP as possible,

#### MVP

- [ ] Create at least one level of the game (deadline: end of February)
  - [ ] Have a health/life system with enemies/traps to keep the game entertaining
  - [ ] Player can jump, move left and right
- [ ] Create the API that will be put into the game so that the player can get the questions (deadline: March 8th)
- [ ] Input API into game and test the game to make sure it works (deadline: March 15th/About a week after API is made)

#### Beyond MVP

- [ ] Create more levels ideally have 9/as much as possible so the player doesn’t get bored of just one level  
  - [ ] Make a coin system, player must collect all coins to complete the game 100% (though not need to win)
  - [ ] Create some levels to be traps only, the player will have to try to get through the level with many (hidden/surprise) traps in the level
